2Gether Website: Production Report

Student: Shandrick de Cuba 163222

This branded website is publicly available at https://shandrickdc.github.io/2gether-PRJ4-website/index.html

Design Elements

  • Vibrant Blue (#0066FF): Creativity, trust, and connectivity.
  • Warm Coral (#FF6B6B): Energetic warmth and CTA highlights.
  • Soft Yellow (#F8D800): Optimism and innovation.
  • Charcoal Gray (#333333): Contrast and professionalism.
  • Light Gray (#F4F4F4): Visual clarity and balance.

Font Choices

  • Montserrat: For headings – modern and bold.
  • Roboto: For body – highly readable and neutral.

UI Patterns

  • Bootstrap Grid System – responsive layout.
  • Sticky Navigation Bar – ease of access.
  • Alternating Text/Image Columns – improved readability.
  • Consistent CTA Buttons – improved user behavior.

Navigation & Content Structure

Navigation: Home – About – Services – Corporate – Contact. Sections include Hero, About, Brand Values, Services, and Contact form. The layout mirrors user behavior using a clear visual hierarchy.

Strategic Fit

The design aligns with 2Gether’s values of inclusiveness, empowerment, and collaboration. It supports the marketing strategy by showcasing tools and community features that empower creatives remotely.

Testing Report

Goals

Test whether users understand the website structure, content, and navigation.

Methods

Interviews with 5 creatives aged 23–40 using both desktop and mobile devices. They explored the site freely and gave feedback.

Results

Positive: Smooth navigation, simple yet effective layout, well-placed CTAs.
Negative: One user noted the mobile font could be larger and suggested hover effects for buttons.

Implemented Improvements

Suggestions were implemented, including mobile font size adjustments and button hover effects.
